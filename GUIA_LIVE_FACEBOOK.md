# Guía rápida para el live de Facebook

Si aparece “Video no disponible”, normalmente no es un error del aplicativo. Facebook bloquea la inserción cuando el video/reel/live no está público, no permite inserción, fue eliminado, cambió de audiencia, exige inicio de sesión o el enlace copiado no corresponde al código de inserción.

## Recomendación para probar

1. Usa preferiblemente YouTube Live para la prueba inicial, porque el iframe suele ser más estable.
2. Si usas Facebook, abre el video desde un computador.
3. Verifica que la publicación esté en público.
4. En el menú del video/reel, busca la opción **Insertar** o **Embed**.
5. Copia el código iframe completo.
6. Pégalo en el panel Super Admin, en **Configurar transmisión**.

## Flujo institucional recomendado

- Transmitir por OBS/StreamYard hacia Facebook y YouTube al mismo tiempo.
- En el aplicativo, usar YouTube como reproductor principal.
- Dejar botón alternativo para abrir Facebook directamente.

Esto evita que la rendición dependa exclusivamente de las restricciones de inserción de Facebook.
