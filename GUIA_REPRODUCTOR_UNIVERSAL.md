# Reproductor universal de transmisión y videos

Esta versión intenta reproducir automáticamente diferentes tipos de enlaces:

- YouTube: watch, youtu.be, live, shorts, embed y playlists.
- Facebook: videos, reels, lives públicos, fb.watch y códigos iframe oficiales.
- Vimeo, Dailymotion y Twitch.
- Google Drive: archivos de video con vista previa.
- Archivos directos: .mp4, .webm, .ogv, .mov.
- Streaming HLS: .m3u8.
- Código iframe pegado directamente.

## Importante

La app puede convertir enlaces y crear el reproductor, pero no puede saltarse bloqueos de plataformas externas. Si Facebook, YouTube u otra plataforma bloquean la inserción por privacidad, país, edad, derechos de autor, inicio de sesión, X-Frame-Options o Content-Security-Policy, el navegador no dejará reproducirlo dentro del aplicativo. En ese caso se muestra el botón “Abrir enlace original”.

## Recomendación para el evento real

Usar YouTube Live como fuente principal, Facebook Live como respaldo y dejar visible el botón externo.
